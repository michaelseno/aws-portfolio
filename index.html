Two flagship systems that showcase cloud architecture depth and reliability-focused SDET execution.

ReliabilityKit (MVP Reliability Toolkit)

Python • Playwright • pytest • CLI • AWS

ReliabilityKit is a local-first reliability execution layer for pytest + Playwright that emits structured run records, failure classifications, and dashboard-grade analytics for faster triage.

Problem

  • Flaky and failed E2E tests are expensive to diagnose without structure.
  • Teams need trend context, not only per-run screenshots and logs.

Architecture and output

  • CLI-first execution + pytest plugin runtime capture
  • Failure digesting, classification, reliability scoring, and trends
  • Scheduled CI + hosted reliability dashboard on AWS S3

TestOrbit (Technical POC)

AWS • React • Serverless • Fargate • Automation

TestOrbit is a technical POC for cloud-native test orchestration. It combines a serverless control plane with containerized execution workers to support API, UI, performance, and security testing workflows.

Architecture focus

  • Serverless API control plane for suite and run lifecycle
  • Fargate execution workers for scalable test runtime isolation
  • Manual, webhook, and scheduled execution paths
  • Credits/results/report retrieval workflow

Status

  • Technical POC with production-oriented patterns
  • Core frontend/backend flows implemented
  • Next: hardening, docs, and architecture case study expansion
