ReliabilityKit (E2E Reliability Toolkit)

Playwright • JUnit5 • Java • CLI

A toolkit that makes E2E failures easier to debug by producing structured run results, HTML reports, and a CLI to open the latest run instantly.

Problem

  • E2E failures are slow to triage (missing artifacts / noisy logs).
  • Teams lose time reproducing failures locally.

What it does

  • Generates results.json and report.html per run
  • Captures screenshot + trace on failure
  • CLI: list runs, open latest report
